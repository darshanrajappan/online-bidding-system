****Online Bidding System****
is a web application project featuring Login, Registration, Dashboard, Create Auction, View Auction and Place Bid modules.
**Technology Stack:**
**Frontend:** HTML, CSS
**Backend:** PHP
**Database:** MySQL
**Other Tools:** SQL for database queries
